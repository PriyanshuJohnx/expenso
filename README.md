# 💸 Expenso (React)

A modern, feature-rich **Expense Tracking Web Application** built using **React (Vite)**.
Track expenses, manage budgets, set goals, and gain insights through interactive analytics.

> 💡 Upgraded from a Vanilla JavaScript implementation to a scalable React architecture.

---

## 🌟 Features

### 📊 Dashboard

* Net Balance, Total Income, Total Expenses, Transactions
* Monthly insights (income, expenses, savings rate)
* Highlights top spending category

---

### 💳 Transactions

* Add, edit, delete transactions
* Categorize expenses (Shopping, Food, Transport, Bills, etc.)
* Real-time updates (no page reload)

---

### 🎯 Budgets

* Set category-wise budgets
* Track spending against limits

---

### 🏆 Goals

* Create financial goals
* Track progress visually

---

### 📈 Analytics

* Pie chart → category-wise spending
* Bar chart → monthly overview
* Built using **Recharts**

---

### 🔍 Search & Filters

* Search transactions instantly
* Filter by category

---

### 🌙 UI Features

* Dark / Light mode toggle
* Clean and responsive design

---

### 📤 Export

* Export data as CSV

---

### 💾 Storage

* LocalStorage persistence
* No backend required

---

## 🛠 Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Charts:** Recharts
* **State:** React Hooks
* **Storage:** LocalStorage

---

## 🧠 Architecture

* Component-based structure
* Custom hooks (`useLocalStorage`)
* Separate folders for components, hooks, utils
* Optimized with `useMemo`

---
